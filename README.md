# MC-Kotlin
A Kotlin Wrapper for Minecraft Development

## Features
- Dependency Injection via Guice
- Simple Command, Event Registration via Annotations: @Command, @Event, @OnEnable, @OnDisable, @OnLoad
- Kotlin Minecraft Coroutines Layer
